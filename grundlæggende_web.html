Grundlæggende web

I Tema 2 arbejdede jeg med de grundlæggende færdigheder, der danner fundamentet for webudvikling og digitalt design.

Temaet gav mig en praktisk og teoretisk introduktion til, hvordan man planlægger, designer og opbygger websites med fokus på brugeroplevelse og struktur. Jeg lærte at lave websites ved hjælp af HTML og CSS hvor jeg arbejdede med semantisk markup, mappestruktur og adskillelse af indhold og styling. Derudover arbejdede jeg med responsivt design ud fra en mobile first-tilgang og anvendte CSS Grid, Flexbox og media queries for at sikre, at layoutet fungerer på tværs af forskellige skærmstørrelser.

I designprocessen arbejdede jeg med Figma til wireframes, styletiles og layout, samt med grundlæggende designprincipper som gestaltprincipper, typografi og designkonventioner. Jeg fik også erfaring med digitale filformater til web, herunder SVG og WebP, samt billede optimering til web brug.

Temaet gav mig desuden indsigt i ophavsret og licenser, og hvordan man anvender billeder og andet indhold korrekt i digitale løsninger. Gennem arbejdet med konkrete opgaver og studiestartsprøven fik jeg erfaring med at omsætte teori til praksis og skabe et responsivt website. Min løsning af opgaven kan ses på billedet til venstre, og ved at klikke på linket nedenfor.
